## üÜò Getting Help

**If training fails:**
1. Check [LIGHTNING_AI_ACTION_PLAN.md](LIGHTNING_AI_ACTION_PLAN.md) troubleshooting section
2. Run `python verify_amharic_training.py`
3. Check you see "[Extended Vocab Fix]" messages
4. Verify loss drops below 3.0 by step 5k

**If resume fails:**
1. Check [COMPLETE_RESUME_TRAINING_GUIDE.md](COMPLETE_RESUME_TRAINING_GUIDE.md) troubleshooting
2. Verify checkpoint exists: `ls -lh trained_ckpts_fixed/latest.pth`
3. Try specific checkpoint: `--resume trained_ckpts_fixed/model_step19000.pth`

**Still stuck?**
- Gather: startup logs, loss values, checkpoint info
- Review all relevant sections in documentation
- Check `knowledge.md` for additional tips
